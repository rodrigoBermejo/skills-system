# Templates de analisis de funnel

---

## Funnel principal: Landing → Activacion

```markdown
# Funnel Analysis: [Nombre]

## Fecha: YYYY-MM-DD

## Etapas

| # | Etapa | Metrica | Valor | Drop-off |
|---|---|---|---|---|
| 1 | Landing visit | Visitantes unicos | N | — |
| 2 | Pricing view | Click en "Precios" | N | X% |
| 3 | Checkout start | Click en "Empezar" | N | X% |
| 4 | Payment complete | Pago exitoso | N | X% |
| 5 | Onboarding start | Inicia wizard | N | X% |
| 6 | Onboarding complete | Completa 5 pasos | N | X% |
| 7 | First conversation | Primer mensaje atendido | N | X% |

## Conversion total: [X%] (etapa 1 → etapa 7)

## Drop-off analysis

### Mayor drop-off: Etapa [N] → [N+1] ([X%])

**Hipotesis:**
1. [Posible causa 1]
2. [Posible causa 2]

**Cambios propuestos:**
1. [Cambio + impacto esperado]
2. [Cambio + impacto esperado]

**Prioridad:** [alta | media | baja]
```

## Matriz de impacto vs esfuerzo

```
              BAJO ESFUERZO    ALTO ESFUERZO
ALTO IMPACTO  | Quick wins  |  Big bets    |
              | Hacer YA    |  Planificar  |
              |-------------|--------------|
BAJO IMPACTO  | Fill-ins    |  Evitar      |
              | Si hay tiempo| No vale pena |
```

Clasificar cada cambio propuesto en esta matriz antes de priorizar.
